Pour en savoir un peu plus sur ce site...

Coucou ! Alors du coup ce site a été créé pour un projet dans une matière que j'ai découvert cette année :

LE DEVELOPPEMENT WEB !

Lors de cette UE, nous avons été introduits à 3 principaux langages utiles au développement web, le

HTML

, le

CSS

et le

JavaScript


Que ce soit pour le HTML, le CSS ou le JavaScript, je n'avais AUCUNE base, je ne connais RIIIIEEEEN du tout! Mais j'ai appris beaucoup de choses tout de même :)




Encore un outil qui m'étais inconnu, VSCode est un éditeur de code très utilisé, mais pas qu'en développement web (on l'a déjà utilisé en Python!) et je le trouve très pratique !

L'idée derrière ce site / Inspiration

Donc, on devait faire ce projet en DevWeb, et en parallèle, on avait ce cours d'Histoire des Arts et des Média dans lequel notre professeure nous a parlé de l'importance d'un

PORTFOLIO

!

C'était alors l'occasion, au travers de ce projet, d'élaborer un premier prototype de portfolio : un site web consacré à mes productions personnelles, où on y trouve des projets que j'ai pu réalisé et qui sont plus ou moins réussis. Et donc ce site en fait totalement partie !


En tout cas, même s'il n'est pas parfaitement réalisé, j'espère qu'il plaira à qui le visitera x)

Utilisations des différents langages : j'ai principalement utilisé HTML pour ce qui est du contenu du site, CSS pour l'affichage (il est joli n'est-ce pas?) et j'ai utilisé JavaScript pour le bouton "+" en page d'accueil et pour le formulaire de contact
